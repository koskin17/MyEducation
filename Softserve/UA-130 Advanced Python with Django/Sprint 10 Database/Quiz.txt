**–û–ø–µ—Ä–∞—Ç–æ—Ä WHERE –≤ SQL:**  
üìå **–û–±—ä—è—Å–Ω–µ–Ω–∏–µ:**  
‚úî **"WHERE" —Ä–∞–±–æ—Ç–∞–µ—Ç –∫–∞–∫ –≥–æ—Ä–∏–∑–æ–Ω—Ç–∞–ª—å–Ω—ã–π —Ñ–∏–ª—å—Ç—Ä**, –ø–æ—Ç–æ–º—É —á—Ç–æ **–æ–Ω –≤—ã–±–∏—Ä–∞–µ—Ç –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–µ —Å—Ç—Ä–æ–∫–∏, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–µ —É—Å–ª–æ–≤–∏—è–º**.  
üìå –ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è `WHERE`:  
```sql
SELECT * FROM users WHERE age > 18;
```
üîπ **–≠—Ç–æ—Ç –∑–∞–ø—Ä–æ—Å –≤—ã–±–µ—Ä–µ—Ç —Ç–æ–ª—å–∫–æ —Å—Ç—Ä–æ–∫–∏, –≥–¥–µ –≤–æ–∑—Ä–∞—Å—Ç (`age`) –±–æ–ª—å—à–µ 18.**  

‚úî **"WHERE" –æ–≥—Ä–∞–Ω–∏—á–∏–≤–∞–µ—Ç –≤–æ–∑–≤—Ä–∞—â–∞–µ–º—ã–µ —Å—Ç—Ä–æ–∫–∏** ‚Üí –ü–æ–∑–≤–æ–ª—è–µ—Ç **—Ñ–∏–ª—å—Ç—Ä–æ–≤–∞—Ç—å –¥–∞–Ω–Ω—ã–µ –Ω–∞ —É—Ä–æ–≤–Ω–µ —Å—Ç—Ä–æ–∫**, –∞ –ù–ï –∫–æ–ª–æ–Ω–æ–∫.  

‚ùå **–ü–æ—á–µ–º—É –¥—Ä—É–≥–∏–µ –≤–∞—Ä–∏–∞–Ω—Ç—ã –Ω–µ–≤–µ—Ä–Ω—ã–µ?**  
- **"a. limits the column data that are returned"** ‚Üí `WHERE` **—Ñ–∏–ª—å—Ç—Ä—É–µ—Ç —Å—Ç—Ä–æ–∫–∏**, –∞ –¥–ª—è –≤—ã–±–æ—Ä–∞ –∫–æ–ª–æ–Ω–æ–∫ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è `SELECT column_name`.  
- **"b. can be used after GROUP BY clause"** ‚Üí –≠—Ç–æ –æ—à–∏–±–∫–∞!  
  - `WHERE` **–∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –ø–µ—Ä–µ–¥ `GROUP BY`**.  
  - –î–ª—è —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏–∏ –ø–æ—Å–ª–µ `GROUP BY` –ø—Ä–∏–º–µ–Ω—è–µ—Ç—Å—è `HAVING`.  

–ö–∞–∫ —Ä–∞—Å—à–∏—Ñ—Ä–æ–≤—ã–≤–∞–µ—Ç—Å—è –∞–±–±—Ä–µ–≤–∏–∞—Ç—É—Ä–∞ SQL?  
üìå **–û–±—ä—è—Å–Ω–µ–Ω–∏–µ:**  
‚úî **SQL (Structured Query Language)** ‚Üí –≠—Ç–æ **—è–∑—ã–∫ –∑–∞–ø—Ä–æ—Å–æ–≤ –∫ –±–∞–∑–∞–º –¥–∞–Ω–Ω—ã—Ö**, –∫–æ—Ç–æ—Ä—ã–π –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è **–∏–∑–≤–ª–µ—á–µ–Ω–∏—è, –∏–∑–º–µ–Ω–µ–Ω–∏—è, —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è –∏ –∞–Ω–∞–ª–∏–∑–∞ –¥–∞–Ω–Ω—ã—Ö** –≤ —Ä–µ–ª—è—Ü–∏–æ–Ω–Ω—ã—Ö –±–∞–∑–∞—Ö –¥–∞–Ω–Ω—ã—Ö.  

üìå **–ü—Ä–∏–º–µ—Ä SQL-–∑–∞–ø—Ä–æ—Å–∞:**  
```sql

```
üîπ –≠—Ç–æ—Ç –∑–∞–ø—Ä–æ—Å **–≤—ã–±–∏—Ä–∞–µ—Ç –≤—Å–µ—Ö –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π, —É –∫–æ—Ç–æ—Ä—ã—Ö –≤–æ–∑—Ä–∞—Å—Ç –±–æ–ª—å—à–µ 18**.  

‚ùå **–ü–æ—á–µ–º—É –¥—Ä—É–≥–∏–µ –≤–∞—Ä–∏–∞–Ω—Ç—ã –Ω–µ–≤–µ—Ä–Ω—ã–µ?**  
- **"Structured Question Language"** ‚ùå ‚Üí –ù–µ–ø—Ä–∞–≤–∏–ª—å–Ω–∞—è —Ñ–æ—Ä–º—É–ª–∏—Ä–æ–≤–∫–∞, —Ç–∞–∫ –∫–∞–∫ SQL **—Ä–∞–±–æ—Ç–∞–µ—Ç —Å –∑–∞–ø—Ä–æ—Å–∞–º–∏ (queries), –∞ –Ω–µ –≤–æ–ø—Ä–æ—Å–∞–º–∏ (questions)**.  
- **"Structuring Query Language"** ‚ùå ‚Üí –í SQL **–Ω–µ—Ç —Å–ª–æ–≤–∞ "Structuring"**, –ø—Ä–∞–≤–∏–ª—å–Ω–æ–µ –Ω–∞–∑–≤–∞–Ω–∏–µ ‚Äî **Structured Query Language**.  
- **"Structural Query Language"** ‚ùå ‚Üí –ë–ª–∏–∑–∫–æ–µ, –Ω–æ **–Ω–µ–ø—Ä–∞–≤–∏–ª—å–Ω–æ–µ –Ω–∞–∑–≤–∞–Ω–∏–µ**, –≤ –æ—Ñ–∏—Ü–∏–∞–ª—å–Ω–æ–º –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏–∏ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è **"Structured"**, –∞ –Ω–µ "Structural".  

–°–∫–æ–ª—å–∫–æ **–ø–µ—Ä–≤–∏—á–Ω—ã—Ö –∫–ª—é—á–µ–π (Primary Key)** –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–æ–∑–¥–∞–Ω–æ –≤ –æ–¥–Ω–æ–π —Ç–∞–±–ª–∏—Ü–µ?  
üìå **–û–±—ä—è—Å–Ω–µ–Ω–∏–µ:**  
‚úî **–¢–æ–ª—å–∫–æ –æ–¥–∏–Ω –ø–µ—Ä–≤–∏—á–Ω—ã–π –∫–ª—é—á (PRIMARY KEY) –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–æ–∑–¥–∞–Ω –≤ —Ç–∞–±–ª–∏—Ü–µ**, –Ω–æ **–æ–Ω –º–æ–∂–µ—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å—Ç–æ–ª–±—Ü–æ–≤ (—Å–æ—Å—Ç–∞–≤–Ω–æ–π –∫–ª—é—á)**.  
üìå **–ü—Ä–∏–º–µ—Ä —Å–æ–∑–¥–∞–Ω–∏—è PRIMARY KEY:**  
```sql
CREATE TABLE users (
    id INT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100) UNIQUE
);
```
üîπ –ó–¥–µ—Å—å `id` —è–≤–ª—è–µ—Ç—Å—è **–µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω—ã–º –ø–µ—Ä–≤–∏—á–Ω—ã–º –∫–ª—é—á–æ–º**, –∫–æ—Ç–æ—Ä—ã–π **—É–Ω–∏–∫–∞–ª—å–Ω–æ –∏–¥–µ–Ω—Ç–∏—Ñ–∏—Ü–∏—Ä—É–µ—Ç –∫–∞–∂–¥—É—é –∑–∞–ø–∏—Å—å**.  

‚ùå **–ü–æ—á–µ–º—É –¥—Ä—É–≥–∏–µ –≤–∞—Ä–∏–∞–Ω—Ç—ã –Ω–µ–≤–µ—Ä–Ω—ã–µ?**  
- **"3" –∏–ª–∏ "2"** ‚ùå ‚Üí –í –æ–¥–Ω–æ–π —Ç–∞–±–ª–∏—Ü–µ **–Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ PRIMARY KEY**, –Ω–æ **–º–æ–∂–Ω–æ –¥–æ–±–∞–≤–∏—Ç—å —Å–æ—Å—Ç–∞–≤–Ω–æ–π –ø–µ—Ä–≤–∏—á–Ω—ã–π –∫–ª—é—á**:  
  ```sql
  PRIMARY KEY (column1, column2)
  ```
- **"infinity"** ‚ùå ‚Üí –ù–µ–≤–æ–∑–º–æ–∂–Ω–æ! **–¢–æ–ª—å–∫–æ –æ–¥–∏–Ω PRIMARY KEY** –º–æ–∂–µ—Ç –±—ã—Ç—å —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω.  

–ö–∞–∫–æ–π –∏–∑ —Å–ª–µ–¥—É—é—â–∏—Ö SQL-–∑–∞–ø—Ä–æ—Å–æ–≤ –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä `SELECT`?  
üìå **–û–±—ä—è—Å–Ω–µ–Ω–∏–µ:**  
‚úî **–ü—Ä–∞–≤–∏–ª—å–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ SELECT:**  
```sql

```
üîπ –≠—Ç–æ—Ç –∑–∞–ø—Ä–æ—Å **–≤—ã–±–∏—Ä–∞–µ—Ç –≤—Å–µ –∫–æ–ª–æ–Ω–∫–∏ (`*`) –∏–∑ —Ç–∞–±–ª–∏—Ü—ã `DataFlair`**.  

‚ùå **–ü–æ—á–µ–º—É –¥—Ä—É–≥–∏–µ –≤–∞—Ä–∏–∞–Ω—Ç—ã –Ω–µ–≤–µ—Ä–Ω—ã–µ?**  
- **a. SELECT col1, col2;** ‚Üí –û—à–∏–±–∫–∞! **–û—Ç—Å—É—Ç—Å—Ç–≤—É–µ—Ç `FROM`**, –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å:  
  ```sql
  SELECT col1, col2 FROM table_name;
  ```
- **b. SELECT # FROM DataFlair;** ‚Üí –û—à–∏–±–∫–∞! **–°–∏–º–≤–æ–ª `#` –Ω–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤ SQL**.  
- **d. SELECT FROM DataFlair;** ‚Üí –û—à–∏–±–∫–∞! **–û—Ç—Å—É—Ç—Å—Ç–≤—É–µ—Ç —Å–ø–∏—Å–æ–∫ –∫–æ–ª–æ–Ω–æ–∫** (`*` –∏–ª–∏ –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–µ –Ω–∞–∑–≤–∞–Ω–∏—è).  

–ö–∞–∫–æ–π –∏–∑ —Å–ª–µ–¥—É—é—â–∏—Ö –≤–∞—Ä–∏–∞–Ω—Ç–æ–≤ **–Ω–µ —è–≤–ª—è–µ—Ç—Å—è –∞–≥—Ä–µ–≥–∞—Ç–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–µ–π** –≤ SQL?  
 **–û–±—ä—è—Å–Ω–µ–Ω–∏–µ:**  
‚úî **SUM, COUNT, AVG** ‚Üí –≠—Ç–æ **–∞–≥—Ä–µ–≥–∞—Ç–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏**, –ø–æ—Ç–æ–º—É —á—Ç–æ –æ–Ω–∏ –≤—ã–ø–æ–ª–Ω—è—é—Ç –æ–ø–µ—Ä–∞—Ü–∏–∏ **–Ω–∞–¥ –º–Ω–æ–∂–µ—Å—Ç–≤–æ–º —Å—Ç—Ä–æ–∫** –∏ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç **–æ–¥–Ω–æ –∞–≥—Ä–µ–≥–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ**.  
üìå **–ü—Ä–∏–º–µ—Ä—ã –∫–æ—Ä—Ä–µ–∫—Ç–Ω—ã—Ö –∞–≥—Ä–µ–≥–∞—Ç–Ω—ã—Ö —Ñ—É–Ω–∫—Ü–∏–π:**  

‚ùå **FIND ‚Äî –Ω–µ —è–≤–ª—è–µ—Ç—Å—è –∞–≥—Ä–µ–≥–∞—Ç–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–µ–π –≤ SQL**  
üîπ –í SQL **–Ω–µ—Ç –≤—Å—Ç—Ä–æ–µ–Ω–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ FIND**. –ï—Å–ª–∏ —Ç—Ä–µ–±—É–µ—Ç—Å—è **–ø–æ–∏—Å–∫ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è**, –∏—Å–ø–æ–ª—å–∑—É—é—Ç `LIKE` –∏–ª–∏ `INSTR()`:  
```sql

```

–ö–∞–∫–∏–µ —Ç–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –≤ SQLAlchemy –¥–ª—è —Å—Ç–æ–ª–±—Ü–æ–≤?  
üìå **–û–±—ä—è—Å–Ω–µ–Ω–∏–µ:**  
‚úî **`DateTime`** ‚Üí –ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è **–¥–∞—Ç—ã –∏ –≤—Ä–µ–º–µ–Ω–∏**.  
üìå **–ü—Ä–∏–º–µ—Ä:**
```python

```

‚úî **`Integer`** ‚Üí –ü—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç **—Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è**.  
üìå **–ü—Ä–∏–º–µ—Ä:**

‚úî **`Numeric`** ‚Üí –ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è **—Ä–∞–±–æ—Ç—ã —Å —á–∏—Å–ª–∞–º–∏ —Å –ø–ª–∞–≤–∞—é—â–µ–π —Ç–æ—á–∫–æ–π**, –∞–Ω–∞–ª–æ–≥–∏—á–Ω–æ `Decimal`.  
üìå **–ü—Ä–∏–º–µ—Ä:**

‚úî **`Boolean`** ‚Üí –•—Ä–∞–Ω–∏—Ç **–∏—Å—Ç–∏–Ω–Ω—ã–µ (`True`) –∏–ª–∏ –ª–æ–∂–Ω—ã–µ (`False`) –∑–Ω–∞—á–µ–Ω–∏—è**.

‚ùå **–ü–æ—á–µ–º—É `Decimal` –ù–ï —è–≤–ª—è–µ—Ç—Å—è –ø—Ä–∞–≤–∏–ª—å–Ω—ã–º –≤–∞—Ä–∏–∞–Ω—Ç–æ–º?**  
üîπ –í SQLAlchemy —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–º —Ç–∏–ø–æ–º —è–≤–ª—è–µ—Ç—Å—è `Numeric`, –∞ **`Decimal` –Ω–∞–ø—Ä—è–º—É—é –Ω–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è**.  

–ö–∞–∫ –≤—ã–≤–µ—Å—Ç–∏ –≤–µ—Ä—Å–∏—é SQLAlchemy?  
üìå **–û–±—ä—è—Å–Ω–µ–Ω–∏–µ:**  
‚úî **`sqlalchemy.__version__`** ‚Üí –≠—Ç–æ **–ø—Ä–∞–≤–∏–ª—å–Ω—ã–π —Å–ø–æ—Å–æ–± –ø–æ–ª—É—á–∏—Ç—å –≤–µ—Ä—Å–∏—é SQLAlchemy**.  
üìå **–ü—Ä–∏–º–µ—Ä –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ–≥–æ –∫–æ–¥–∞:**

‚ùå **–ü–æ—á–µ–º—É –¥—Ä—É–≥–∏–µ –≤–∞—Ä–∏–∞–Ω—Ç—ã –Ω–µ–≤–µ—Ä–Ω—ã–µ?**  
- **`print sqlalchemy.version`** ‚ùå ‚Üí –í SQLAlchemy **–Ω–µ—Ç —Ç–∞–∫–æ–≥–æ –∞—Ç—Ä–∏–±—É—Ç–∞**.  
- **`print sqlalchemy.ver`** ‚ùå ‚Üí –û—à–∏–±–∫–∞, **–Ω–µ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç `ver` –≤ SQLAlchemy**.  
- **`print ver()`** ‚ùå ‚Üí –ù–µ—Å—É—â–µ—Å—Ç–≤—É—é—â–∞—è —Ñ—É–Ω–∫—Ü–∏—è –≤ SQLAlchemy, –≤—ã–∑–æ–≤–µ—Ç –æ—à–∏–±–∫—É.  

–ö–∞–∫–∞—è –º–æ–¥–µ–ª—å –±–∞–∑—ã –¥–∞–Ω–Ω—ã—Ö –æ–ø–∏—Å—ã–≤–∞–µ—Ç –¥–∞–Ω–Ω—ã–π –∫–æ–¥?  
‚úî **–≠—Ç–æ—Ç –∫–æ–¥ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç –±–∞–∑–æ–≤—É—é –º–æ–¥–µ–ª—å (Basic model) –≤ SQLAlchemy**.  
üîπ –ö–ª–∞—Å—Å **`BaseModel`** –Ω–∞—Å–ª–µ–¥—É–µ—Ç `Base`, –Ω–æ **–Ω–µ –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω –¥–ª—è –Ω–µ–ø–æ—Å—Ä–µ–¥—Å—Ç–≤–µ–Ω–Ω–æ–≥–æ —Å–æ–∑–¥–∞–Ω–∏—è —Ç–∞–±–ª–∏—Ü—ã**.  
üîπ **–û–Ω –∞–±—Å—Ç—Ä–∞–∫—Ç–Ω—ã–π** (`__abstract__ = True`), —Ç–æ –µ—Å—Ç—å –µ–≥–æ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–∞–∫ –æ—Å–Ω–æ–≤—É –¥–ª—è –¥—Ä—É–≥–∏—Ö –º–æ–¥–µ–ª–µ–π.  
üîπ **–û–ø—Ä–µ–¥–µ–ª–µ–Ω—ã —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–µ –ø–æ–ª—è:**  
   - `id` ‚Üí **–ü–µ—Ä–≤–∏—á–Ω—ã–π –∫–ª—é—á** (—Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω—ã–π, —É–Ω–∏–∫–∞–ª—å–Ω—ã–π, –∞–≤—Ç–æ–∏–Ω–∫—Ä–µ–º–µ–Ω—Ç).  
   - `created_at`, `updated_at` ‚Üí **–û—Ç–º–µ—Ç–∫–∏ –≤—Ä–µ–º–µ–Ω–∏** (`TIMESTAMP`).  
üîπ **–í–∫–ª—é—á—ë–Ω –º–µ—Ç–æ–¥ `__repr__`**, –∫–æ—Ç–æ—Ä—ã–π –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å—Ç—Ä–æ–∫–æ–≤–æ–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –æ–±—ä–µ–∫—Ç–∞.  

‚ùå **–ü–æ—á–µ–º—É –¥—Ä—É–≥–∏–µ –≤–∞—Ä–∏–∞–Ω—Ç—ã –Ω–µ–≤–µ—Ä–Ω—ã–µ?**  
- **Separate table** ‚ùå ‚Üí –ö–æ–¥ **–Ω–µ —Å–æ–∑–¥–∞—ë—Ç –æ—Ç–¥–µ–ª—å–Ω—É—é —Ç–∞–±–ª–∏—Ü—É**, –∞ —Å–ª—É–∂–∏—Ç **–æ—Å–Ω–æ–≤–æ–π –¥–ª—è –¥—Ä—É–≥–∏—Ö –º–æ–¥–µ–ª–µ–π**.  
- **Not tables** ‚ùå ‚Üí –ö–æ–¥ **–æ—Ç–Ω–æ—Å–∏—Ç—Å—è –∫ ORM**, –≥–¥–µ **–º–æ–¥–µ–ª–∏ –ø—Ä–∏–≤—è–∑–∞–Ω—ã –∫ —Ç–∞–±–ª–∏—Ü–∞–º –±–∞–∑—ã –¥–∞–Ω–Ω—ã—Ö**.  
- **Index** ‚ùå ‚Üí –í –∫–æ–¥–µ **–Ω–µ—Ç –∏–Ω–¥–µ–∫—Å–æ–≤** (`Index`, `Unique`, `ForeignKey`), –æ–Ω –∑–∞–¥–∞—ë—Ç –±–∞–∑–æ–≤—É—é —Å—Ç—Ä—É–∫—Ç—É—Ä—É –º–æ–¥–µ–ª–∏.  

–ö–∞–∫–∏–µ –º–µ—Ç–æ–¥—ã –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –¥–ª—è –∑–∞–∫—Ä—ã—Ç–∏—è —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è —Å –±–∞–∑–æ–π –¥–∞–Ω–Ω—ã—Ö –≤ SQLAlchemy? (–í—ã–±–µ—Ä–∏ –¥–≤–∞)  
üìå **–û–±—ä—è—Å–Ω–µ–Ω–∏–µ:**  
‚úî **`result.close()`** ‚Üí –≠—Ç–æ—Ç –º–µ—Ç–æ–¥ –∑–∞–∫—Ä—ã–≤–∞–µ—Ç **–∫—É—Ä—Å–æ—Ä—ã (ResultSet)** –ø–æ—Å–ª–µ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –∑–∞–ø—Ä–æ—Å–∞.  
üìå **–ü—Ä–∏–º–µ—Ä:**  
```python

```

‚úî **–ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ –∑–∞–∫—Ä—ã—Ç–∏–µ –ø–æ—Å–ª–µ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è SQL-–∑–∞–ø—Ä–æ—Å–∞** ‚Üí SQLAlchemy **–∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ —É–ø—Ä–∞–≤–ª—è–µ—Ç —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è–º–∏** –ø—Ä–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ `Session()`.  
üîπ **–ï—Å–ª–∏ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è `with Session()`**, —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ **–∑–∞–∫—Ä—ã–≤–∞–µ—Ç—Å—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏**:  
```python
from sqlalchemy.orm import sessionmaker

Session = sessionmaker(bind=engine)
with Session() as session:
    result = session.execute("SELECT * FROM users")
    # –ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ –∑–∞–∫—Ä—ã—Ç–∏–µ –ø–æ—Å–ª–µ –∑–∞–≤–µ—Ä—à–µ–Ω–∏—è –±–ª–æ–∫–∞ "with"
```

‚ùå **–ü–æ—á–µ–º—É –¥—Ä—É–≥–∏–µ –≤–∞—Ä–∏–∞–Ω—Ç—ã –Ω–µ–≤–µ—Ä–Ω—ã–µ?**  
- **`close`** ‚ùå ‚Üí –ù–µ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –º–µ—Ç–æ–¥–∞ `close()` –±–µ–∑ –æ–±—ä–µ–∫—Ç–∞ (`result.close()` –ø—Ä–∞–≤–∏–ª—å–Ω—ã–π).  
- **`end()`** ‚ùå ‚Üí –ù–µ—Ç —Ç–∞–∫–æ–≥–æ –º–µ—Ç–æ–¥–∞ –≤ SQLAlchemy –¥–ª—è –∑–∞–∫—Ä—ã—Ç–∏—è —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–π.  
- **`sqlalchemy.exit()`** ‚ùå ‚Üí –í SQLAlchemy **–Ω–µ—Ç –≤—Å—Ç—Ä–æ–µ–Ω–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ exit()**.
